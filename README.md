# dcmsort Package:

The goal of `dcmsort` is to provide wrapper functions to sort DICOM
images

## Installation

You can install `dcmsort` from GitHub with:

``` r
# install.packages("remotes")
remotes::install_github("muschellij2/dcmsort")
```
